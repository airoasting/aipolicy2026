# AI 정책 카탈로그 2026

**v1.0.0 · 2026-06-07**

2026년 6·3 제9회 전국동시지방선거에서 당선된 **16개 시·도지사**의 AI 공약 **108건**을, 출처·신뢰도와 함께 한곳에 정리한 웹사이트. 단일 HTML(`index.html`) + 데이터(`data.js`)로 동작한다.

## 무엇이 들어있나
- **Executive Summary(요약)** — 108개 공약의 쏠림과 빈칸을 4개 카드로 진단(투자 경쟁 쏠림 / 비어 있는 자리 / 규모는 키웠지만 관건은 실행 등).
- **한눈에 보기(비교 격자)** — 16개 시·도 × 8개 분야 분포 매트릭스. 색이 진할수록 공약 수가 많고, 좌→우로 쏠림이 보인다.
- **건수 지도** — 지역별 공약 수를 대한민국 지도 + 순위 리스트로. 정당색(민주 파랑 / 국민의힘 빨강) 기준.
- **지역별 / 카테고리별 카드** — 당선자 사진·주요 경력·**득표율** + AI 정책 담당 조직(확인/추정/미발표) + 정책 카드(분야·정량 수치·신뢰도 배지·**팩트체크 배지**·출처 링크). 본문은 불릿로 정리.
- 화면 **우측 하단 고정 ‘요약’ 버튼**(어디서든 요약으로 이동), 검색, 지역/카테고리 보기 전환, 모바일 최적화.

## 데이터
- `data.js`의 `window.AI_DATA` — 16개 지역, 정책 108건. 멀티 에이전트 파이프라인(지역 리서치 → 적대적 팩트검증)으로 수집·검증.
- **득표율**: 16명 당선인 전원 중앙선거관리위원회 공식 확정치로 전수 검증.
- 각 정책 `factcheck`: `url_live`(링크 생존) · `cross_verified`(복수매체 교차) · `redteam_survived`(반증 시도 생존).
- 신뢰도 태그: 직접인용 / 보도확인 / 단일출처 / 확인안됨 / 예비후보기.
- AI 담당 인물·조직은 취임 전이라 다수가 ‘미발표/추정’ — 날조 없이 정직 표기.

## 실행 방법
ES 모듈 + importmap을 쓰므로 `file://` 직접 열기 대신 **로컬 서버**로 실행한다.

```bash
# 프로젝트 루트에서
node server.js
# 브라우저에서 http://localhost:8123 접속
```

> 데이터 갱신 시 `data.js`의 `window.AI_DATA` 배열만 교체하면 된다. 사이트 코드는 데이터 드리븐이라 정책 개수가 바뀌어도 매트릭스·지도·카드가 자동 적응한다.

- 당선인 16인 사진 `.webp` 교체, 경력·임기 정리(국회의원·도지사 종료연도 보정, ‘민선 N기’ 표기 제거).
