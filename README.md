# 민선 9기 광역단체장 AI 정책 지도 (3D 스크롤리텔링)

2026.06.03 제9회 전국동시지방선거 당선 **16개 시·도지사**의 AI 정책을, 스크롤 다운하며 3D로 보는 데스크탑 웹사이트.

## 무엇이 들어있나
- **인트로 → 16개 지역 → 아웃트로** 스크롤 플라이스루. 정당색(민주 파랑 / 국민의힘 빨강) 발광 글래스 타일을 카메라가 하강하며 통과.
- 각 지역 카드: **당선자 사진·소속·주요 경력 + AI 정책 담당 인물(확인/추정/미발표) + 정책 카드**(카테고리·정량 수치·신뢰도 배지·**팩트체크 배지**·출처 링크).
- 좌측 점 레일 내비게이션(클릭 점프), 밝은 배경, 데스크탑(폭 1200px+ 권장).

## 데이터
- `data.js` — 16개 지역, 정책 108건. 멀티 에이전트 파이프라인(지역 리서치 → 적대적 팩트검증)으로 수집·검증. 2026-06-07 RED 팩트 재검증: 미검증/반증·취약 21건 삭제(106→85) 후, ≥2개 독립출처로 교차확인된 신규 공약 23건 보강(85→108).
- 각 정책 `factcheck`: `url_live`(링크 생존) · `cross_verified`(복수매체 교차) · `redteam_survived`(반증 시도 생존).
- 신뢰도 태그: 직접인용 / 보도확인 / 단일출처 / 확인안됨 / 예비후보기.
- AI 담당 인물은 취임 전이라 다수가 '미발표/추정' — 날조 없이 정직 표기.

## 실행 방법 (데스크탑 브라우저)
ES 모듈 + importmap을 쓰므로 `file://` 직접 열기 대신 **로컬 서버**로 실행하세요.

```bash
# 프로젝트 루트에서
node server.js
# 브라우저에서 http://localhost:8123 접속
```

> 데이터 갱신 시 `data.js`의 `window.AI_DATA` 배열만 교체하면 됩니다. 사이트 코드는 데이터 드리븐이라 정책 개수가 바뀌어도 3D·카드가 자동 적응합니다.
